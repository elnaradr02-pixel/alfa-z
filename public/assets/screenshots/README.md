# Скриншоты «примеры работ» для CodeWindow

Положите сюда PNG/WebP скриншоты демо-уроков. Я подключу их как `image` в компонент
`CodeWindow` (тёмная рамка редактора) на страницах курсов и в блоке «примеры работ».

Рекомендуемые имена (чтобы я сразу привязал к курсам) — но подойдут любые, просто скажите какой к какому курсу:

| Файл | Что на скрине | Курс |
|---|---|---|
| `backend-bot-simulator.png` | Симулятор Telegram-бота «попробуй сам» | Бэкенд на Python |
| `backend-guess-game.png` | Игра «Угадай число» (if/elif/else) | Бэкенд на Python |
| `web-box-styler.png` | Стилизация `.box` (живой предпросмотр CSS) | Веб-разработка |
| `web-primary-switcher.png` | Переключатель Primary-цвета | Веб-разработка |
| `mobile-app-generator.png` | «Одна строка → приложение» (квиз-генератор) | Мобильная разработка |
| `mobile-flutter-scaffold.png` | Конструктор Flutter Scaffold | Мобильная разработка |

Формат: PNG или WebP, ширина ~1000–1400px. Соотношение сторон не критично — рамка обрежет
аккуратно (object-cover 4:3). Для геймдева скрина пока нет — если есть, добавьте `gamedev-*.png`.
